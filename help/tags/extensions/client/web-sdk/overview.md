# Übersicht über die Tag-Erweiterung &quot;Adobe Experience Platform Web SDK&quot;

Die Tag-Erweiterung &quot;Adobe Experience Platform Web SDK&quot; sendet Daten von Web-Eigenschaften über Adobe Experience Platform Edge Network an die Adobe Experience Cloud. Die Experience Platform Web SDK-Erweiterung ermöglicht das Streaming von Daten in Platform, das Synchronisieren von Identitäten, das Opt-in und die automatische Erfassung von Kontextdaten.

Unter [Erste Schritte](getting-started.md) erfahren Sie, wie Sie diese Tag-Erweiterung einrichten und bereitstellen, um mit der Datenerfassung zu beginnen.

Sehen Sie sich das folgende Video an, um die [!DNL Web SDK] und [!DNL Edge Network] von Adobe Experience Platform in Aktion zu sehen, wobei ein einziger Aufruf zum Senden von Daten an [!DNL Experience Platform], [!DNL Analytics], [!DNL Audience Manager] und [!DNL Target] verwendet wird.
