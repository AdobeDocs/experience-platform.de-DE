# Aktionstypen

Auf dieser Seite werden alle unterstützten Aktionen innerhalb einer Tag-Regel aufgelistet. So erstellen oder bearbeiten Sie die Aktionen innerhalb einer Tag-Regel:

1. Melden Sie sich mit Ihren Adobe ID[Anmeldeinformationen bei ](https://experience.adobe.com)experience.adobe.com) an.
1. Navigieren Sie zu **[!UICONTROL Data Collection]** > **[!UICONTROL Tags]**.
1. Wählen Sie die gewünschte Tag-Eigenschaft aus.
1. Navigieren Sie zu **[!UICONTROL Rules]** und wählen Sie dann die gewünschte Regel aus.
1. Wählen Sie unter [!UICONTROL Actions] eine vorhandene Aktion aus oder erstellen Sie eine Aktion.
1. Legen Sie im Dropdown-Feld [!UICONTROL Extension] den Wert **[!UICONTROL Adobe Experience Platform Web SDK]** fest und wählen Sie dann die gewünschte [!UICONTROL Action Type] aus.

Die Tag-Erweiterung „Web SDK&quot; bietet die folgenden Aktionen:

* [**[!UICONTROL Apply propositions]**](apply-propositions.md): Rendern von Vorschlägen in Single-Page Applications ohne Inkrementierung von Metriken.
* [**[!UICONTROL Apply response]**](apply-response.md): Führen Sie eine Aktion basierend auf einer Antwort von Edge Network durch.
* [**[!UICONTROL Evaluate rulesets]**](evaluate-rulesets.md): Manueller Trigger einer Regelsatzauswertung.
* [**[!UICONTROL Get Media Analytics tracker]**](get-media-analytics-tracker.md): Exportieren Sie die alte Media-API in ein Fensterobjekt.
* [**[!UICONTROL Redirect with identity]**](redirect-with-identity.md): Ermöglicht die Freigabe einer Besucherkennung über Domains hinweg, deren Inhaber Sie sind.
* [**[!UICONTROL Send event]**](send-event.md): Sendet Ereignisdaten an die Edge Network.
* [**[!UICONTROL Send media event]**](send-media-event.md): Sendet Mediendaten an die Edge Network.
* [**[!UICONTROL Set consent]**](set-consent.md): Legt das gewünschte Einverständnis für den Besucher fest.
* [**[!UICONTROL Update variable]**](update-variable.md): Ändert ein [Variable](../data-element-types.md#variable)-Datenelement.
