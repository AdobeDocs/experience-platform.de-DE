# PDPA (Thailand)-Terminologie

Im Folgenden finden Sie eine Liste von Begriffsbestimmungen für Schlüsselbegriffe im Zusammenhang mit dem Gesetz zum Schutz personenbezogener Daten (PDPA) in Thailand.

| Begriff | Definition |
| --- | --- |
| **Persönliche Daten** | Informationen, die eine Person direkt oder indirekt identifizieren und identifizieren, mit Ausnahme von Informationen über eine verstorbene Person. |
| **Datencontroller** | Eine Person oder Organisation, die befugt ist, Entscheidungen über die Erhebung, Nutzung oder Weitergabe personenbezogener Daten zu treffen. |
| **Datenprozessor** | Eine Person oder Organisation, die personenbezogene Daten gemäß den Anweisungen des für die Verarbeitung Verantwortlichen erhebt, verwendet oder offenlegt. |
| **Sensible Daten** | Personenbezogene Daten, die sich auf rassische oder ethnische Herkünfte, politische Meinungen, religiöse oder philosophische Überzeugungen, sexuelles Verhalten, Strafregister, Gesundheitsdaten, Behinderung, Informationen zur Vereinigung des Handels, genetische Daten, biometrische Daten oder alle Daten beziehen, die die betroffene Person betreffen können. |
