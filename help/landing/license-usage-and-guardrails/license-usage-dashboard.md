# [!UICONTROL Dashboard zur Lizenzverwendung] {#license-usage-dashboard}

Die Adobe Experience Platform-Benutzeroberfläche bietet ein Dashboard, über das Sie wichtige Informationen zur Lizenznutzung Ihres Unternehmens anzeigen können, wie sie in einer täglichen Momentaufnahme erfasst werden.

Detaillierte Anweisungen zum Zugriff auf das Dashboard zur Lizenznutzung in der Benutzeroberfläche und zur Interaktion damit sowie weitere Informationen zu den verfügbaren Metriken, die im Dashboard angezeigt werden, finden Sie im [Dashboard-Handbuch zur Lizenznutzung](../../dashboards/guides/license-usage.md).

Einen Überblick über alle Dashboard-Funktionen innerhalb von Experience Platform erhalten Sie im Abschnitt [Übersicht über Dashboards](../../dashboards/home.md).
