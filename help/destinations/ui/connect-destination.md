# So stellen Sie eine Verbindung zu einem Ziel her

Gehen Sie wie folgt vor, um in der Echtzeit-Kundendatenplattform eine Verbindung zu einem Ziel herzustellen:

1. Navigieren Sie in der linken Navigationsleiste zu **[!UICONTROL Ziele]**.
2. Wählen Sie auf der Registerkarte [[!UICONTROL Katalog]](./destinations-workspace.md#catalog) das Ziel aus, zu dem Sie eine Verbindung herstellen möchten.
3. Klicken Sie auf **[!UICONTROL Ziel verbinden]**, um den Zielverbindungsassistenten zu starten.
4. Eine schrittweise Anleitung zum Herstellen einer Verbindung mit bestimmten Zielen finden Sie auf den Zielseiten. Siehe [Cloud-Datenspeicherung-Ziele](../catalog/cloud-storage/workflow.md), [E-Mail-Marketing-Ziele](../catalog/email-marketing/overview.md)und [Werbeziele](../catalog/advertising/overview.md).
