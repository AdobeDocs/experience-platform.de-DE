# (Beta) Apache HDFS-Anschluss

Adobe Experience Platform bietet native Konnektivität für Cloud-Anbieter wie AWS, Google Cloud Platform und Azurblase, sodass Sie Ihre Daten von diesen Systemen mitnehmen können. Ingetierte Daten können als JSON, Parquet oder als Trennzeichen formatiert werden. Zu den Cloud-Datenspeicherung-Anbietern zählen Apache HDFS.

In der folgenden Dokumentation finden Sie Informationen zum Verbinden von Apache HDFS mit Plattform mithilfe von APIs oder der Benutzeroberfläche:
