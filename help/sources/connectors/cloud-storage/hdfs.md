# (Beta) HDFS-Anschluss

Adobe Experience Platform bietet native Konnektivität für Cloud-Anbieter wie AWS, Google Cloud Platform und Azurblase, sodass Sie Ihre Daten von diesen Systemen mitnehmen können. Ingetierte Daten können als JSON, Parquet oder als Trennzeichen formatiert werden. Zu den Cloud-Datenspeicherung-Anbietern zählen HDFS.

Die nachstehende Dokumentation enthält Informationen zum Verbinden von HDFS mit Plattform mithilfe von APIs:
