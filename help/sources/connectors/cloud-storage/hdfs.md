# (Beta) Apache HDFS-Anschluss

>[!NOTE]
>Der Apache HDFS-Anschluss befindet sich in der Beta-Version. Weitere Informationen zur Verwendung von Beta-gekennzeichneten Connectors finden Sie in der Übersicht [zu den](../../home.md#terms-and-conditions) Quellen.

Adobe Experience Platform bietet native Konnektivität für Cloud-Anbieter wie AWS, Google Cloud-Platform und Azurblase, sodass Sie Ihre Daten von diesen Systemen übertragen können. Ingetierte Daten können als JSON, Parquet oder als Trennzeichen formatiert werden. Zu den Cloud-Datenspeicherung-Anbietern zählen Apache HDFS.

Die nachstehende Dokumentation enthält Informationen zum Verbinden von Apache HDFS mit der Platform mithilfe von APIs oder der Benutzeroberfläche:
