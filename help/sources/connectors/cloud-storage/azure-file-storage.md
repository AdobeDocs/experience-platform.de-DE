# (Beta) Blue File Datenspeicherung Connector

>[!NOTE]
>Der Azurblase File Datenspeicherung Stecker ist in Beta. Weitere Informationen zur Verwendung von Beta-gekennzeichneten Connectors finden Sie in der Übersicht [zu den](../../home.md#terms-and-conditions) Quellen.

Adobe Experience Platform bietet native Konnektivität für Cloud-Anbieter wie AWS, Google Cloud-Platform und Azurblase, sodass Sie Ihre Daten von diesen Systemen übertragen können.

Cloud-Datenspeicherung-Quellen können Ihre eigenen Daten in die Platform bringen, ohne dass Sie sie herunterladen, formatieren oder hochladen müssen. Ingetierte Daten können als XDM-JSON-, XDM-Parkett oder als Trennzeichen formatiert werden. Jeder Schritt des Prozesses ist in den Sources-Workflow integriert. Mit Platform können Sie Daten aus der Datenspeicherung von &quot;Blue File&quot;in Batches importieren.
