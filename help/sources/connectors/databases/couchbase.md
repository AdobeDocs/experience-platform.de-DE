# (Beta) CouchBase-Connector

Adobe Experience Platform bietet native Konnektivität für Datenbankanbieter wie Microsoft, MySQL und Azurblase, sodass Sie Ihre Daten von diesen Systemen übertragen können. Es werden verschiedene Arten von Drittanbieter-Datenbanken unterstützt, einschließlich relationaler Datenbanken, NoSQL-Datenbanken oder Data Warehouse. Die Unterstützung für Datenbankanbieter umfasst auch CouchBase.

Die nachstehende Dokumentation enthält Informationen dazu, wie Sie mithilfe von APIs eine Verbindung zwischen CouchBase und Platform herstellen:
