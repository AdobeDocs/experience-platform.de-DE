# (Beta) Oracle Connector

>[!NOTE]
>Der Oracle Connector befindet sich in der Betaphase. Weitere Informationen zur Verwendung von Beta-gekennzeichneten Connectors finden Sie in der Übersicht [zu den](../../home.md#terms-and-conditions) Quellen.

Adobe Experience Platform bietet native Konnektivität für Datenbankanbieter wie Microsoft, MySQL und Azurblau. Sie können Ihre Daten aus diesen Systemen in Platform bringen.

Es werden verschiedene Arten von Drittanbieter-Datenbanken unterstützt, einschließlich relationaler Datenbanken, NoSQL oder data warehouse. Zu den Datenbankanbietern zählen Oracle.

Die nachstehende Dokumentation enthält Informationen zum Verbinden von Oracle mit der Platform mithilfe von APIs oder der Benutzeroberfläche:
