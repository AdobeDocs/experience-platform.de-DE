# (Beta) Azurblauer Data Explorer-Anschluss

>[!NOTE]
>Der Azurblauer Data Explorer-Anschluss befindet sich in der Betaphase. Weitere Informationen zur Verwendung von Beta-gekennzeichneten Connectors finden Sie in der Übersicht [zu den](../../home.md#terms-and-conditions) Quellen.

Adobe Experience Platform bietet native Konnektivität für Datenbankanbieter wie Microsoft, MySQL und Azurblau. Sie können Ihre Daten aus diesen Systemen in Platform bringen.

Es werden verschiedene Arten von Drittanbieter-Datenbanken unterstützt, einschließlich relationaler Datenbanken, NoSQL oder data warehouse. Zur Unterstützung von Datenbankanbietern gehört auch Azurblase Data Explorer.

Die nachstehende Dokumentation enthält Informationen dazu, wie Sie den Blue Data Explorer mit APIs oder der Benutzeroberfläche mit der Platform verbinden:
