# [!DNL Deltashare]

Sie können den [!DNL Deltashare]-Quell-Connector verwenden, um eine sichere Verbindung zu [!DNL Databricks Delta Sharing]-Servern herzustellen und virtuelle Datensätze in Adobe Experience Platform zu erstellen. Mit dieser Funktion können Sie virtuelle Datensätze abfragen, externe Tabellen überprüfen und Experience-Datenmodell (XDM)-Schemata zuordnen und sie in allen Experience Platform-Programmen und -Services nutzen. Durch [!DNL Deltashare] können Sie die physische Aufnahme von Datensätzen vermeiden und so Duplizierungen, Speicherkosten und den Betriebsaufwand reduzieren.
