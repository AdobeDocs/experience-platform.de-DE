# Datenflüsse - Übersicht

In Adobe Experience Platform werden Daten aus einer Vielzahl von Quellen erfasst, innerhalb der Experience Platform analysiert und zu einer Vielzahl von Zielen aktiviert. Plattform erleichtert die Verfolgung dieses potenziell nicht-linearen Datenflusses durch Transparenz mit Datenflüssen.

Datenflüsse sind eine Darstellung von Datenaufträgen, die Daten über die Plattform verschieben. Diese Datenflüsse werden über verschiedene Dienste konfiguriert und unterstützen Sie beim Verschieben von Daten von den Quellschnittstellen zu den Zielgruppen-Datensätzen, zum Identitäts- und Profil-Dienst und zu Zielen.

## Überwachen von Datenflüssen

Die Überwachung von Datenflüssen kann entweder über die Plattform-APIs oder die Plattform-Benutzeroberfläche erfolgen. Informationen zum Überwachen von Datenflüssen mithilfe der API finden Sie im [Monitoring DataFlows API-Lernprogramm](./api/monitor.md). Um zu erfahren, wie Sie Datenflüsse in der Plattform-Benutzeroberfläche überwachen, lesen Sie bitte entweder den [Überwachungsdataflows für das Quelltutorial](./ui/monitor-sources.md) oder den [Überwachungsdataflows für das Tutorial](./ui/monitor-destinations.md)Ziele.
