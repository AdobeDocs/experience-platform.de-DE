# Ad-hoc-Schemata

In specific circumstances, it may be necessary to create an [!DNL Experience Data Model] (XDM) schema with fields that are namespaced for usage only by a single dataset. Dies wird als Ad-hoc-Schema bezeichnet. Ad-hoc-Schemata werden in verschiedenen Workflows zur Datenerfassung verwendet, einschließlich der Erfassung von CSV-Dateien und der Erstellung bestimmter Arten von Quell-Connectors. For more information on ad-hoc schemas, including steps on how to create one in the [!DNL Schema Registry] API, please refer to the [ad-hoc schema tutorial](../tutorials/ad-hoc.md).
