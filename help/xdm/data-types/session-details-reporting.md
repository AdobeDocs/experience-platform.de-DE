# [!UICONTROL Sitzungsdetails] Berichtdatentyp

[!UICONTROL Sitzungsdetails] -Berichterstellung ist ein standardmäßiger XDM-Datentyp (Experience-Datenmodell), der Daten im Zusammenhang mit Medienwiedergabesitzungen verfolgt.
Medienberichtsfelder werden von Adobe-Diensten verwendet, um die von Benutzern gesendeten Mediensammlungsfelder zu analysieren. Diese Daten werden zusammen mit anderen spezifischen Benutzermetriken berechnet und in Berichten verwendet. Das Schema umfasst eine breite Palette von Eigenschaften, die Einblicke in das Benutzerverhalten und die Inhaltskonsummuster bieten. Verwenden Sie den Datentyp [!UICONTROL Sitzungsdetails] zur Erfassung der Benutzerinteraktion durch Protokollierung von Wiedergabeereignissen, Anzeigeninteraktionen, Fortschrittsmarken, Pausen und anderen Metriken.
