# [!UICONTROL Session Details] Reporting-Datentyp

[!UICONTROL Session Details] Reporting ist ein standardmäßiger Experience-Datenmodell (XDM)-Datentyp, mit dem Daten im Zusammenhang mit Medienwiedergabesitzungen verfolgt werden.
Medien-Reporting-Felder werden von Adobe-Services verwendet, um die von Benutzenden gesendeten Mediensammlungsfelder zu analysieren. Diese Daten werden zusammen mit anderen spezifischen Benutzermetriken berechnet und in Berichten erfasst. Das Schema umfasst eine breite Palette von Eigenschaften, die Einblicke in das Benutzerverhalten und die Verbrauchsmuster von Inhalten bieten. Verwenden Sie den Datentyp &quot;[!UICONTROL Session Details] Reporting“, um Benutzerinteraktionen zu erfassen, indem Sie Wiedergabeereignisse, Anzeigeninteraktionen, Fortschrittsmarken, Pausen und andere Metriken protokollieren.
