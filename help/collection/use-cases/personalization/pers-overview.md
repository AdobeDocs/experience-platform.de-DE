# Übersicht über Personalization-Anwendungsfälle

Mit Adobe Experience Platform Web SDK können Unternehmen eine Vielzahl von Personalisierungsanwendungsfällen für ihre Web-Eigenschaften bereitstellen. Sie können Datenerfassungs-, Entscheidungs- und Aktivierungsprozesse nahtlos mit Adobe-Lösungen wie Target, Journey Optimizer und Audience Manager verbinden.

SDK unterstützt flexible Architekturen, einschließlich Client-, Server- und Hybridmodellen, sodass Sie personalisierte Angebote abrufen und rendern können, die den Anwendungsfällen der meisten Unternehmen entsprechen.

## Beispiele für Web SDK

Zusätzlich zu den Dokumentseiten in diesem Ordner verwaltet Adobe ein Repository mit Beispielanwendungen, auf die Sie verweisen können. Unter [Beispiele für Web-SDK](https://github.com/adobe/alloy-samples/) auf GitHub finden Sie weitere Anwendungsfälle für die Personalisierung, darunter:

* Client-seitige Personalisierung
* Server-seitige Personalisierung
* Hybride Personalisierung
* Personalization mit Single Page Applications
