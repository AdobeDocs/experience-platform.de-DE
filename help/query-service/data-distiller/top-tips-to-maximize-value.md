# Wichtigste Tipps zur Wertmaximierung mit Adobe Experience Platform Data Distiller

Diese Seite enthält den Beispieldatensatz, damit Sie das anwenden können, was Sie in der Sitzung „OS656 - Top-Tipps zur Wertmaximierung mit Adobe Experience Platform Data Distiller&quot; gelernt haben. Sie erfahren, wie Sie Implementierungen von Adobe Real-time Customer Data Platform und Journey Optimizer beschleunigen können, indem Sie Echtzeit-Kundenprofildaten anreichern. Diese Anreicherung nutzt tiefe Einblicke in Kundenverhaltensmuster, um Zielgruppen für die Bereitstellung und Optimierung von Erlebnissen zu erstellen.

Mithilfe einer Fallstudie analysieren Sie die Verhaltensdaten der Benutzer und erstellen ein Modell für Aktualität, Häufigkeit, Währung (RFM) - eine Marketing-Analysetechnik für die Kundensegmentierung basierend auf Kaufmustern.
