# Dynamisches Segment in der Experience Platform-Benutzeroberfläche erstellen

Das folgende Video soll Sie bei der Erstellung dynamischer Segmente unterstützen. Es zeigt ein Beispiel für ein Kundenerlebnis und eine kurze Übersicht über die wichtigsten Funktionen im [!DNL Platform] -Schnittstelle. Weitere Informationen zur Segmentierung und zum Erstellen von Segmenten in der Benutzeroberfläche finden Sie in der [Segment Builder-Handbuch](../ui/segment-builder.md).

Wenn Sie ein Segment mithilfe der Segmentation-API erstellen möchten, lesen Sie bitte den Abschnitt [Segment-API-Handbuch erstellen](../tutorials/create-a-segment.md).

>[!NOTE]
>
>Die Benutzeroberfläche von Experience Platform wird häufig aktualisiert und kann sich seit der Aufzeichnung dieses Videos geändert haben. Aktuelle Informationen finden Sie im Abschnitt [Segment Builder-Handbuch](../ui/segment-builder.md).
