# Erstellen eines dynamischen Segments in der Experience Platform-Benutzeroberfläche

Im folgenden Video erfahren Sie, wie Sie dynamische Segmente erstellen. Es zeigt ein Beispiel für ein Kundenerlebnis, gefolgt von einer kurzen Übersicht über die wichtigsten Funktionen in der [!DNL Platform]. Weitere Informationen zur Segmentierung und zum Erstellen von Segmenten in der Benutzeroberfläche finden Sie im [Segment Builder-Handbuch](../ui/segment-builder.md).

Wenn Sie daran interessiert sind, ein Segment mithilfe der Segmentierungs-API zu erstellen, lesen Sie bitte das [Handbuch zur Erstellung einer Segment-API](../tutorials/create-a-segment.md).

>[!NOTE]
>
>Die Benutzeroberfläche von Experience Platform wird häufig aktualisiert und kann sich seit der Aufzeichnung dieses Videos geändert haben. Die aktuellsten Informationen finden Sie im [Segment Builder-Handbuch](../ui/segment-builder.md).
