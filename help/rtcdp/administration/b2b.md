# Zugriffssteuerung in Adobe Real-time Customer Data Platform B2B edition

Die Zugriffssteuerung für Real-Time CDP B2B edition wird über die [Adobe Admin Console](https://adminconsole.adobe.com) verwaltet. Diese Funktion nutzt Produktprofile in [!DNL Admin Console], um Benutzern Berechtigungen und Sandboxes zuzuordnen. Mithilfe dieser Funktion können Admins den Zugriff auf bestimmte Real-Time CDP-Funktionen wie die B2B-Benutzeroberfläche für definierte Benutzergruppen gewähren oder einschränken.

Weitere Informationen zur Zugriffssteuerung finden Sie unter [Zugriffssteuerung - Übersicht](../../access-control/home.md) in der [!DNL Experience Platform].

In den folgenden Dokumenten finden Sie detaillierte Anweisungen zur Verwendung von [Adobe Admin Console](https://adminconsole.adobe.com) zum Aktivieren des Zugriffs auf die B2B-Benutzeroberfläche für ein Produktprofil:

* [Aktivieren der B2B-Benutzeroberfläche für ein neues Produktprofil](../../access-control/ui/create-profile.md)
* [Aktivieren der B2B-Benutzeroberfläche für ein vorhandenes Produktprofil](../../access-control/ui/details-and-services.md)
