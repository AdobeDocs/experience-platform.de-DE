# Quellen in Real-time Customer Data Platform

[!DNL Adobe Real-Time Customer Data Platform] ermöglicht Ihnen die Aufnahme von Daten aus externen Quellen und bietet Ihnen die Möglichkeit, die eingehenden Daten mithilfe von [!DNL Platform]-Services zu strukturieren, zu kennzeichnen und anzureichern. Sie können Daten aus einer Vielzahl von Quellen aufnehmen, z. B. aus Adobe-Apps, Cloud-basiertem Speicher, Software von Drittanbietern, Ihrem CRM-System und mehr.

Einen detaillierteren Überblick über Quellen, einschließlich einer Liste der verfügbaren Quellen und Links zu Tutorials, erhalten Sie in der [Quellendokumentation](../../sources/home.md).
