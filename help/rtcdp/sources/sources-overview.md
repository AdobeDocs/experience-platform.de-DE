# Quellen in Real-time Customer Data Platform

[!DNL Adobe Real-Time Customer Data Platform] ermöglicht die Aufnahme von Daten aus externen Quellen und bietet Ihnen die Möglichkeit, die eingehenden Daten mithilfe von [!DNL Platform]-Services zu strukturieren, zu kennzeichnen und zu verbessern. Daten können Sie aus verschiedenen Quellen erfassen, z. B. aus Adobe-Lösungen, Cloud-basiertem Speicher, Software von Drittanbietern und Ihrem CRM-System.

Eine detailliertere Übersicht über die Quellen, einschließlich einer Liste der verfügbaren Quellen und Links zu Tutorials, finden Sie im Abschnitt [Quelldokumentation](../../sources/home.md).
