# Segment Builder in Real-Time Customer Data Platform

[!DNL Adobe Real-Time Customer Data Platform] basiert auf Adobe Experience Platform und kann alle Segment Builder-Funktionen nutzen, die Teil von [!DNL Experience Platform] sind. In der Benutzeroberfläche von Segment Builder können Sie neue Zielgruppen erstellen, auf vorhandene Zielgruppen zugreifen und vorhandene Zielgruppen ändern.

Eine detailliertere Anleitung zum Segment Builder finden Sie im [Segment Builder-Benutzerhandbuch](../../segmentation/ui/segment-builder.md).
